# Место

## Описание проекта:
Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.
## Технологии:
1. HTML5;
2. CSS3;
3. JavaScript;
4. БЭМ Nested;
5. Flexbox;
6. Grid layout;
7. Работа с DOM;
8. Адаптивная верстка;
9. Семантические тэги.
